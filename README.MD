## MikroTik RouterOS Plugin for Checkmk

This plugin provides comprehensive monitoring of MikroTik RouterOS devices using both API v2 (for RouterOS 7.x+) and SNMP for standard metrics.  
It includes checks for:

- BGP and OSPF neighbors
- VRRP status
- PSU, Fan, and Temperature sensors
- IPsec tunnels
- License expiration and file age
- Firewall rules based on comment filters
- System health and crashfile detection

Please read the CHANGELOG.MD file to discover all the new features and changes made to the plugin.
